# R-6: Linux AppImage target is documented but not configured

## Approach

Added `src-tauri/tauri.linux.conf.json` with a Linux-specific bundle target override for `["appimage"]`. Tauri automatically merges platform-specific configuration files into the main config, and arrays are replaced under JSON Merge Patch semantics, so this keeps the macOS `["app", "dmg"]` targets from R-5 while making Linux builds request the documented AppImage artifact.
