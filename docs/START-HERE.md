# Start here

Open this folder in Grok, Codex, Claude Code, or Antigravity. Say what you want. **`AGENTS.md` loads automatically.**

## Questions

Ask anything. You get words back. The repo does not change.

## Work

Ask for something to be built or fixed. The agent follows `AGENTS.md` and reports what to try when done.

## If unclear to the agent

It should ask: *"Explanation only, or change the repo?"* You don't need to remember that — it's the default protocol.
